机器学习 $h_\theta (x)$

如图
如图

我们的目的就是要找出一个函数$h_\theta (x)$ 然后通过函数 $h_\theta (x)$，我们就能预测出我们想要的结果。 比如ng视频经常提到的预测房价的函数, 癌症判断函数等等。未来找到这样的函数。我们需要应用很多步骤和数学技巧。这篇里面主要记录回归问题。

$h_\theta (x)$ 函数上的红点是训练数据的样本。也就是说 我们通过相对少量(对整个函数无数个点来说 红点算少了)样本，训练出最终函数$h_\theta (x)$

Classification

$h_\theta (x) = g(\theta ^Tx)$
$z = \theta ^Tx$
$g(z) = \frac{1}{1+e^{-z}}$
g(z)的图

从图片上看
若 $z\to +\infty$;
则 $g(z)\to 1$。
若 $z\to -\infty$;
则 $g(z)\to 0$。

真假判断 $y =1$; $y=0$。 $h_\theta (x) = 1$; $h_\theta (x) = 0$

对于一般分类来说 只存在二种情况 y=1, y=0。 但是我们根据函数$h_\theta (x)$可知它的范围
$ 0< h_\theta (x) < 1。$
为了使得 $h_\theta (x) = 1$ $h_\theta (x) = 0$
存在k($0<k<1$)
当$h_\theta (x) > k$，则 $h_\theta (x) = 1$
当$h_\theta (x) < k$，则 $h_\theta (x) = 0$
k的取值一般情况看实际项目。一般情况下取 $k = 0.5 $。当 k = 0.5的时候 根据上面图像和g公式 我们可以知道 $0.5 = \frac{1}{1+e^{-z}}$, 可以推断出 z = 0。则可以得出下面的几个结论
$ \theta ^Tx >=0 \Rightarrow\ y=1$
$ \theta ^Tx <0 \Rightarrow\ y=0$
至此 我们可知 通过控制 $\theta ^Tx$ 就可以控制 $h_\theta (x)$的真假。

Cost Function

$J(\theta ) = \frac{1}{m}\sum_{i=1}^{m}Cost(h_\theta(x^{(i)}), y^{(i)})
$
$Cost(h_\theta(x),y) = -log(h_\theta(x))$; 当 $ y = 1 $
$Cost(h_\theta(x),y) = -log(1-h_\theta(x))$; 当 $ y = 0 $
log正常函数的图像如下
则 $log(z)$ 图像如下
那么 $-log(z)$的图像相当于 $log(z)$图像相对z轴对称 所以$-log(z)$如图最终求出 $Cost(h_\theta(x),y) = -log(h_\theta(x))$图像如下

$Cost(h_\theta(x),y) = -log(1-h_\theta(x))$; 函数图像如下

绘图有1个隐藏条件 我们绘制 $log(h_\theta(x))$函数图像的时候 log 的底数是 > 1的。也就是 a > 1。

总结上2图，得出
当$y=0$且$h_\theta(x) \to 1$ 那么$ Cost(h_\theta(x), y) \to \infty $
当$y=1$且$h_\theta(x) \to 0$ 那么$ Cost(h_\theta(x), y) \to \infty $

因为y的不同值会产生2种不同的 $J(\theta)$，但是这2个函数可以合并成一个表达式
$Cost(h_\theta(x), y) = -y log(h_\theta(x)) - (1-y)log(1-h_\theta(x))$
$J(\theta ) = -\frac{1}{m}\sum_{i=1}^{m}[y^{(i)} log(h_\theta(x^{(i)})) + (1-y^{(i)})log(1-h_\theta(x^{(i)})) ]$
如果是向量的实现代码如下
$h = g(X\theta)$
$J(\theta) = \frac{1}{m}.(-y^T log(h) - (1-y)^T log(1-h))$

Gradient Descent

$\theta_j := \theta_j - \alpha \frac{\vartheta}{\vartheta\theta_j} $
计算公式如下
$\theta_j := \theta_j - \frac{\alpha}{m}\sum_{i=1}^{m}(h_\theta(x^{(i)}) - y^{(i)})x_j^{(i)} $
如果用向量实现
$\theta := \theta - \frac{a}{m}X^T(g(X \Theta) - \vec{y})$

决策线 $\theta$ 间距之间的关系

  • $\theta$ 与 决策界是 90度正交的。通过这个特效 可以知道P的间距。
  • 当 $\theta$ = 0 的时候，决策线是经过原点的。
  • p dot || $\theta$ || >= 1 , p dot || $\theta$ || <= -1 这些情况下。我们的目的是使得$\theta$较小，那么根据这些情况 只能 p 变大。 重图像来看，就是 p 的距离会变大，通过 p的距离可以调整决策线的画法。(p 可以表示正样本和负样本距离决策线的距离。
